Html-Writr by html-writr

Write to Html Pages Quickly and Easily

Html writr is an innovative system that allows users with little to no technical knowledge to edit static html files. It is very easy to setup, and requires no database to use. The editing experience is simplistic but full featured, using http://www.nicedit.com for the wysiwyg editor, and allowing users to edit the page title, meta description, and meta keywords. Preparing html files for html writer to edit is very quick, just a couple of lines of code and you're done.

Install

1. Download the most recent version of html writr. 2. Open the zip file, and copy the contents of the download (except for index.html) to the root of where you want html files to be editable- most often inside public_html. 3. Go to yourwebsite.com/path/to/writr/writr_install.php and fill out the information with a username and password that you will use to login to edit. 4. Click install, and html writr should be installed, and you should be logged in.
